## 项目运行
```js
git clone https://github.com/anno-er/Daily-Record.git
npm install
npx react-native start
a
```

## 项目介绍
基于React Native + react-native-android-widgt 的一个安卓平台的桌面打卡应用，主要功能包括登录、计划添加、桌面打卡widget
